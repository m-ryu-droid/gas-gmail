 Gmail下書き一斉作成ツール

このGASは、Googleスプレッドシートの宛先リストをもとに、Gmailの下書きを1人1通ずつ作成します。
このコード自体はメールを送信しません。

## 方針

- 送信者は、このGASを実行したGoogleアカウントです。
- 宛先ごとに個別のGmail下書きを作成します。
- 受信者同士のメールアドレスは見えません。
- 会社名と氏名を本文に差し込みます。
- 下書き作成済みの行は二重作成しません。
- 添付ファイルは必要な場合だけ設定できます。

## 使い方

1. Googleスプレッドシートを新規作成します。
2. メニューから `拡張機能 > Apps Script` を開きます。
3. `gas_gmail_draft_sender.gs` の内容を貼り付けて保存します。
4. スプレッドシートを再読み込みします。
5. メニューに `メール下書き作成` が出ます。
6. `メール下書き作成 > 初期設定シートを作成` を実行します。
7. `宛先リスト`、`設定`、`本文テンプレート` を入力します。
8. まず `テスト下書き作成` を実行します。
9. Gmailの下書きフォルダで内容を確認します。
10. 問題なければ `本番下書き作成` を実行します。
11. Gmailの下書きフォルダで各メールを確認して、手動で送信します。

## 宛先リスト

| 列名 | 内容 |
| --- | --- |
| 送信対象 | TRUEの行だけ対象です |
| 会社名 | `{{会社名}}` に差し込まれます |
| 氏名 | `{{氏名}}` に差し込まれます |
| メールアドレス | 宛先です |
| 件名 | 行ごとに件名を変えたい場合に入力します |
| ステータス | GASが自動入力します |
| 下書き作成日時 | GASが自動入力します |
| 下書きID | GASが自動入力します |
| エラー内容 | GASが自動入力します |

## 設定

| 項目 | 内容 |
| --- | --- |
| テスト送信先 | テスト下書きの宛先です |
| 共通件名 | 件名列が空の場合に使われます |
| 差出人名 | 空欄で問題ありません。表示名を変えたい場合だけ入力します |
| 添付ファイルID | 添付するGoogleドライブファイルのIDです。複数の場合はカンマ区切りです |

## 本文テンプレート

`本文テンプレート` シートの `A2` に本文を書きます。

使用できる差し込み項目:

```text
{{会社名}}
{{氏名}}
```

例:

```text
{{会社名}}
{{氏名}} 様

いつもお世話になっております。

本文をここに入力します。

よろしくお願いいたします。
```

## 添付ファイルIDの確認方法

Googleドライブで添付したいファイルを開き、URLの中にあるIDをコピーします。

例:

```text
https://drive.google.com/file/d/ここがファイルID/view
```

設定シートの `添付ファイルID` にそのIDを入れると、全ての下書きに添付されます。

## 注意

- 本番下書き作成を実行しても、その時点では送信されません。
- Gmailの下書きフォルダで内容を確認してから、送信担当者が手動で送信してください。
- 送信担当者のアカウントでApps Scriptを承認、実行してください。
- `下書き作成済み` の行は再作成されません。再作成したい場合はステータス、下書き作成日時、下書きIDを空にしてください。
